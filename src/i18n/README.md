# Translations

Edit translation text in:

- `src/i18n/locales/en.ts`
- `src/i18n/locales/fr.ts`

Keep the same keys in both files. Project card descriptions use the pattern:

- `project.<projectId>.summary`
- `project.<projectId>.needs`
- `project.<projectId>.licenseNote`

The selected language is saved in `localStorage` under `ai-home-lab:language:v1`.
